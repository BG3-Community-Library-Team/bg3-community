So, you got a great idea for a new custom race for Baldur's Gate 3. Great! The only problem is, you have no idea how to actually make a custom race.

Sound like you? Well you're in the right place.

The Template

EmeraldTechno has created a handy race template to go along with this tutorial. You can download the template here: https://github.com/EmeraldTechnopath/CMTY-Race-Template

Just visit the link, click the green Code button, and then click Download ZIP.

Unzip the zip file, delete the README.md, and you're ready to start following the tutorial.
