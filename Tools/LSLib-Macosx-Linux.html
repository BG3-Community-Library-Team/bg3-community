Extracting the contents of a .pak file on MacOSX

 

Note: This guide assumes a basic understanding of how to use CLI commands inside Terminal.
All commands listed should be run inside Terminal.

  • 1- Install brew
  • 2- Once brew is installed, install `wine`
  • 3- Install .net
  • 4- Download the Extraction tools: 
  • 5- Unzip the ExportTool .zip file:
  • 6- Run ConverterApp.exe
  • 7- Extract the .pak file

1- Install brew

Install instructions:

Homebrew

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"


2- Once brew is installed, install `wine`

brew install wine-stable


3- Install .net

These tools all require .net to run. So you’ll need to download and install the windows version of .net and run the installer inside win.

Download from Download .NET 8.0 Desktop Runtime (v8.0.14) - Windows x64 Installer
And then run the installer with `wine`:

wine windowsdesktop-runtime-8.0.14-win-x64.exe


4- Download the Extraction tools: Releases · Norbyte/lslib

Click on the ExportTool-v1.19.5.zip to download it.
 

5- Unzip the ExportTool .zip file:

unzip ExportTool-v1.19.5.zip

For whatever reason this creates a folder called “Packed” with all the binaries inside it:

unzip -l ExportTool-v1.19.5.zip |grep exe

6- Run ConverterApp.exe

cd Packed
wine ConverterApp.exe

Should look like this:
 

7- Extract the .pak file

Click on the “PAK/LSV Tools” table and select your Input .pak file and your output location and click the “Extract Package” button.

You’ll get a popup letting you know it worked.
