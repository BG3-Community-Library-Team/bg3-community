Glossary

Shaders are what determines how an object is rendered, what texture maps it allows and the parameters that can be utilized.

Shaders come in two types: VirtualTextures Shaders which are encoded textures, and Non-Virtual Textures Shaders.

The pathing of shaders can be found in this entry:

<attribute id="SourceFile" type="LSString" value="Public/Shared/Assets/Materials/Characters/CHAR_BASE.lsf" />

Explanation

  • Filename: The name of the shader in the actual game files.
  • Alpha: The shader's capacity to allow transparency, partial or fully. Two-Sided Alpha edits the model to appear as double-sided.
  • MSKCloth: The shader's capacity to make an object tintable, corresponds each colour to a parameter.
  • VertCut: The shader's capacity to implement masking, which is used to hide parts of models that overlap. (such as helmets hiding hair)
  • GlowMap: The shader's capacity to allow for an object to glow.
  • Physics: The Shader's capacity to contain physics (Mostly used in Atmosphere Shaders for flowers)
