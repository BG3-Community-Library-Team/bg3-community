Description

  • Filename: The name of the model in the actual game files.
  • UUID: The unique identifier for the model.
  • Sub-Mesh: The individual meshes within a GR2 file.
  • Material ID: The UUID that contains the texture information for each sub-mesh.

Cloth meshes typically (though not always) use the Material ID 9e2966c7-b61c-4bc1-bef1-a79cb5fde067 (Engine_Default_Mesh_Lit_Opaque). It is safe to use this engine default for any cloth mesh, but the ID of the other mesh will also work!
