Explanation

  • Filename: The name of the beard in the actual game files.
  • Image: An image of the beard.
  • In-game name: The name of the beard in Character Creation.
  • UUID: The unique identifier for the beard.
  • HairPresetResourceID: The identifier for the beard preset, which controls things like wind movement, frizz, etc.
  • MaterialID: The identifier for the material/texture the beard uses.
  • SkeletonResource: The identifier for the skeleton resource, only used in beards with physics.
  • Autosnap: Whether the beard applies to all races/genders or only one race/gender.
