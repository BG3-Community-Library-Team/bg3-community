Explanation

  • Filename: Name of the piercing mesh in the game files.
  • Image: Image of the piercing mesh.
  • UUID: Unique identifier of the piercing.
  • VisualResource: Unique identifier of the piercing's VisualBank entry.
  • MaterialID: Unique identifier of the piercing's material.
  • PiercingLocation: Visual locator attachment point the piercing is set up for in vanilla.

Please note that most meshes exist for both the left and right side. Both versions of the same mesh will be listed in the same entry.
